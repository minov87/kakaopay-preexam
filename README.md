## Kakaopay 서버개발 사전과제

> Rest API 기반 쿠폰시스템 개발

### 개발 프레임워크
* Java 1.8
* Spring Boot 2.2.6
* JPA

### 문제해결 전략
#### 필수 문제 
- [ ] 랜덤 쿠폰 N개 생성 API
- [ ] 사용자에게 쿠폰 지급 API 
- [ ] 사용자 지급 쿠폰 조회 API
- [ ] 지급된 쿠폰 사용 API
- [ ] 지급된 쿠폰 사용 취소 API
  - [ ] 쿠폰 사용 취소시 만료된 쿠폰의 처리 방안 강구
- [ ] 당일 만료 쿠폰 목록 조회 API
